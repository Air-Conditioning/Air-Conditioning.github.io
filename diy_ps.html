P5js Drawing Program (Fall 2024)

This is a simple drawing program that I coded in P5js. I wanted to make a somewhat versatile drawing program, but I also considered a theme I could create the program around. I enjoy illustrating dragons in my free time, so I chose to include dragon-related elements like a brush to paint stacks of coins and red rubies.

Keybinds:

1 - Black line tool

2 - Red brush tool

3 - Orange brush tool

4 - Yellow brush tool

5 - Green brush tool

6 - Blue brush tool

7 - Purple brush tool

8 - Square fire tool

9 - Gray spray tool

0 - Random color rectangle tool

Z - Eraser

G - Gem brush tool

H - Coin stack brush tool
